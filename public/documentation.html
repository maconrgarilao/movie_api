Movie API Documentation

Business Logic URL HTTP Method Request body data format Response body data format
Get list of movies /movies GET None A JSON object holding data about all the movies
Get data about a movie by Title /movies/[Title]/ GET None A JSON object holding data about a movie containing a title and director.
Example:
{
Title: [Name],
Genre: [Type],
Director: [Name],
}
Get data about a movie by Genre /movies/[Title]/[Genre]/ GET None A JSON object holding data about a movie containing a title and director.
Example:
{
Title: [Name],
Genre: [Type],
Director: [Name],
}
Get data about a movie by Director /movies/directors/[DirectorName]/ GET None A JSON object holding data about a director.
Example:
{
Director Name: [Name],
Bio: [Description],
}
Add a movie /movies POST A JSON object holding data about a movie to add A JSON object holding data about a movie that was added
Remove a movie by title /movies/[Title]/ DELETE None A text message indicating whether the movie was succesfully removed from their list of favorites
Create username /users POST None A JSON object of a new user and id
Remove a user /user/[id]/ DELETE None A text message indicating whether the user has been removed
